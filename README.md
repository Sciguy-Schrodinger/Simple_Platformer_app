# 🏃 Platformer

A classic platformer game built with **Pygame**.  
Jump between platforms, collect coins, and avoid enemies!

---

## 🎮 Features

- **Classic platformer mechanics** — run, jump, and explore
- **Coin collection** — collect coins to score points
- **Enemy AI** — enemies patrol the platforms
- **Multiple platforms** — three platforms at different heights
- **Smooth animations** — left, right, jump, and death animations
- **Sound effects** — coin collection, death, and background music
- **Score tracking** — keep track of your progress
- **Dynamic difficulty** — enemies respawn with each coin collected

---

## 🛠️ Technologies Used

- **Python 3** — core logic
- **Pygame** — 2D rendering, input handling, audio

---

## 🚀 How to Run

### Requirements

Install dependencies:

bash
pip install pygame

## Run

python3 platformer.py

🎮 Controls
Key	Action
Left Arrow	Move left
Right Arrow	Move right
Up Arrow	Jump
Close Window	Exit the game

🧠 How It Works

   1. Player — moves with arrow keys and jumps on platforms

   2. Platforms — three platforms at different heights

   3. Gravity — the player falls when not on a platform

   4. Coins — collect coins to increase your score

   5. Enemies — patrol the platforms and move randomly

   6. Collision — touching an enemy ends the game

   7. Win condition — collect coins to progress
